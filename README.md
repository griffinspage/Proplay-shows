# Proplay-shows
Proplay shows is a sleek movie website that allows movie  enthusiasts to review a movie by watching a trailer first b4 actually downloading it. It essentially saves time and gives a  good overview of a movie.
I built Proplay Shows as a movie preview hub where I can quickly discover the latest films and TV shows. With this site I’m able to search for any movie, explore trending and top‑rated titles, and instantly watch official trailers right inside the app. It’s a sleek, interactive project designed to give movie lovers a simple way to preview what’s hot before deciding where to watch.

✨ Features
🔍 Search Movies – Find any movie by title using the TMDb API.

📈 Trending & Popular – Browse what’s hot right now.

⭐ Top Rated – Explore critically acclaimed films.

🎭 Genre Browsing – Action, Drama, Horror, and more.

▶️ Trailer Playback – Click a movie card to watch its official trailer.

🎨 Responsive UI – Clean design with smooth scrolling and hover effects.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

API: TMDb API for movie data and trailers

Design: Custom CSS animations, Material Symbols icons

🚀 Getting Started
Prerequisites
A modern web browser (Chrome, Edge, Firefox, Safari).

A free TMDb API key (sign up at TMDb (themoviedb.org in Bing)).

Installation
Clone the repository:

bash
git clone https://github.com/yourusername/proplay-shows.git
Navigate into the project folder:

bash
cd proplay-shows
Open index.html in your browser.

Add your own TMDb API key in java.js:

javascript
const API_KEY = "YOUR_TMDB_API_KEY";
📌 Notes
This project is for educational/demo purposes.

TMDb provides trailers and metadata, but not full movie streams.

All trailers are embedded legally via YouTube.
